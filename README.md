# 동국홀딩스 2026년 팀별 목표등록 시스템

웹에서 팀을 선택하고 목표(연두보고과제/운영과제)를 등록·저장하는 시스템입니다.

## 실행 방법

1. 가상환경 권장 후 의존성 설치:
   ```bash
   pip install -r requirements.txt
   ```

2. 서버 실행:
   ```bash
   python app.py
   ```

3. 브라우저에서 접속:
   - http://127.0.0.1:5000

## 구성

- **메인 화면**: 전략팀, 재경팀, 커뮤니케이션팀, 윤리경영팀, 인사기획팀, 비서팀 버튼
- **목표 등록 화면**: 목표구분(연두보고과제/운영과제), 과제명, 과제설명, 담당자(팀원 복수 선택)
- **팀원 데이터**: `teammate.xlsx` (컬럼: team, name, jikwi)
- **저장 데이터**: `goals.json` (같은 폴더에 자동 생성)

## 파일 설명

- `app.py` - Flask 웹 앱 (라우트, API, 팀원 로드, 목표 저장)
- `objectmanagement.py` - 팀 목록·경로 설정
- `templates/` - 메인·목표등록 HTML
- `teammate.xlsx` - 팀별 팀원 명단
